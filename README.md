## About
An introduction to Python and writing Python code using a Jupyter Notebook.  Work with common data types in Python.  Use variables to store data, and use lists, tuples, dictionaries and sets to organize data.  Understand what a Boolean is, and where it is useful.

## Learning Objectives

*You will be able to:*

- Learn to use basic Jupyter Notebook features.
- Define integers, strings, tuples, lists, and dictionaries.
- Demonstrate arithmetic operations and string operations.
